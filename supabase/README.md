1. Create a Supabase project.
2. Open SQL Editor, run [`schema.sql`](/z:/book-maind/book-main/supabase/schema.sql).
3. Run [`seed.sql`](/z:/book-maind/book-main/supabase/seed.sql) for sample data.
4. Copy [`.env.example`](/z:/book-maind/book-main/.env.example) to `.env` and fill keys.
5. Install deps: `npm install`.

Main client is in [`lib/supabase.ts`](/z:/book-maind/book-main/lib/supabase.ts).
